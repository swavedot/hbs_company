Waypoints란?

스크롤 이벤트를 감지해서 특정 요소가 화면의 특정 위치에 도달했을 때
클래스를 추가하거나, 애니메이션을 실행하거나, 함수를 실행하는 트리거를 걸 수 있게 해주는 방식
 
 - 트리커(trigger) : 특정 조건이 만족될 때 자동으로 실행되는 이벤트 또는 동작

Waypoints 대표적 두가지 방식

1. Waypoints.js(jQuery 기반 라이브러리) : 간단한 문법으로 스크롤 트리거 효과 만들 수 있음
2. Intersection Observer(Js표준 방식) : 브라우저 기본  API jQuery 없이도 사용 가능
